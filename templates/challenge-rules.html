The 32 Deck Challenge is a popular Magic: The Gathering deckbuilding goal where players create one Commander deck for each of the 32 possible color combinations. This includes all mono-colored decks, two-color pairs, three-color combinations, four-color combinations, five-color, and colorless.

‚öñÔ∏è Challenge Rules

  1. One Deck Per Color Combination

    You must build exactly one deck for each of the 32 color combinations.

  2. No Card Duplication (Except Basic Lands)

    Each non-basic land card can only appear in ONE deck across your entire collection. Basic lands (Plains, Island, Swamp, Mountain, Forest, Wastes, and snow-covered versions) are exempt and can be used in multiple decks.

  3. Standard Commander Format Rules Apply

    All decks must follow the official Commander format rules:

    • Exactly 100 cards (including your commander)
    • One legendary creature or planeswalker as commander
    • Singleton format (only one copy of each card, except basic lands)
    • Cards must match commander's color identity
    • No banned cards
  4. Color Identity Restriction

    All cards in a deck must match the color identity of the commander. A card's color identity includes all mana symbols in its mana cost and rules text.

üí° Tips & Strategies

  • Plan ahead: Think about which cards work in multiple color combinations and allocate them strategically.
  • Budget wisely: You'll need 3,200+ unique cards total. Consider budget alternatives.
  • Theme diversity: Try different strategies (aggro, control, combo, tribal) across your decks to keep things interesting.
  • Track progress: Use this app's visual grid to see which combinations you've completed and which need work.

üö´ Common Mistakes to Avoid

  • Using the same powerful non-basic land (like Command Tower) in multiple decks
  • Building all your favorite commanders first and running out of good cards for the rest
